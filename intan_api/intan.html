Module contents

Intan: Python interface for reading, processing, and visualizing data from Intan Technologies’ RHD2000 and RHS2000 acquisition systems.

This package includes modules for: - Reading and parsing .rhd and .rhs files - Streamed acquisition via TCP/IP from the Intan RHX software - Visualization of high-density EMG or LFP signals - Configuration and device control
